# MotorPH Payroll System

## Description
A simple Java Swing payroll management application for MotorPH.  
This project includes employee management, attendance tracking, and payroll computation features with a graphical user interface.

## Features
- Employee Form
- Attendance Form
- Payroll Form
- Attendance Hour Calculation
- Payroll Calculation
- Input Validation
- User-Friendly Interface

## Technologies Used
- Java
- Java Swing
- NetBeans IDE

## How to Run
1. Open the project in NetBeans
2. Run `MainFrame.java`
3. Use the menu buttons to access the forms
